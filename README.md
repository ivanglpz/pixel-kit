# Pixel Kit

Pixel Kit is a React-based image editor inspired by tools like Photoshop and Figma.  
Built with Next.js, Konva, Jotai, and TypeScript.

- Modern tools for editing images in the browser.
- Open source and easy to extend.

## Getting Started

Install dependencies and start the development server:

```sh
pnpm install
pnpm dev
```
